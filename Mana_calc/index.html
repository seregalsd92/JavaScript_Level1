Калькулятор маны для игры Empires & Puzzles.

Данный калькулятор показывает, когда зарядится маной герой в обороне.

По горизонтали расположено необходимое число кубиков, попавших в героя.

По вертикали - необходимое число ходов, которое должно пройти до зарядки героя.

Посчитайте коэффициент ускорения маны от отряда, талантов, семьи и прочих бонусов. Введите его в графу для коэффициента ускорения маны. Например 1,15 (с 30-ым отрядом). Выберите скорость набора маны героем из списка. Нажмите кнопку "посчитать" или Enter, после установки значения в одном из полей.
